### Sign Message via One-time Code

This shows how you can sign a message using one-time codes.

<Note>This only demonstrates frictioneless wallet-signing flow. You are not signing/sending any transactions.</Note>

1. Go to [actioncode.app](https://actioncode.app) 
2. Get a code and submit here with the message you want to sign
3. Then confirm signing in actioncode.app
