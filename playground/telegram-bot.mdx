---
title: 'Telegram Bot'
description: 'Sign message or transaction directly in Telegram with Action Codes'
---

A Telegram bot that lets users test **Action Codes** **directly from their wallet**. No website, no wallet connect required.

---

<Steps>
  <Step title="Start the bot">
    Go to [@action_codes_bot](https://t.me/action_codes_bot) on Telegram  
    Tap **Start** then type command `/message`
  </Step>
  <Step title="Put a message to sign">
    Reply with a message you want to sign
  </Step>
  <Step title="Go to actioncode.app in your browser or wallet browser">
    Get a one-time code
  </Step>
  <Step title="Enter one-time Action Code">
    Submit one-time code by replying in Telegram
  </Step>
  <Step title="Approve signing in actioncode.app">
    You will see telegram chat outputs signed message
  </Step>
</Steps>
