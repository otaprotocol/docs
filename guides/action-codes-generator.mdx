title: 'Action Code Generator'
description: 'How to get one-time Action Code'
---

The easiest way to get an one-time secure Action Code is using our official hosted Action Code Generator at [actioncode.app](https://actioncode.app)

1. Visit [actioncode.app](https://actioncode.app)
2. Connect your wallet (we recommend using Solana wallets' in-app browser)
3. Copy/share codes with anyone, anywhere
4. To inspect code and validate you can use [Action Code Web Inspector Tool](https://relay.ota.codes/inspect)

Also we encourage to try out [Action Codes Playground](/playground).
